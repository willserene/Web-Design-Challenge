Latitude vs Four Weather Variables

This dashboard reflects the findings from a prior weather analysis of over 500 randomly selected cities. The latitude of each city was plotted against four variables including the maximum temperature (F), humidity (%), cloudiness (%), and wind speed (mph) in order to analyze the respective relationships.

The creation of this dashboard entailed the use of HTML, CSS, and Bootstrap to create a total of seven websites which include this landing page, four plots of the respective analyses, a comparison of all of the plots, and lastly a page reflecting the data used to build them.
